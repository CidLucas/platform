# Vizu RAG Factory

This library provides a factory for creating Retrieval-Augmented Generation (RAG) runnables and tools for the Vizu application.

## Overview

The Vizu RAG Factory simplifies the process of creating and configuring RAG-based tools. It uses `langchain-core` and other internal Vizu libraries to create custom RAG chains that can be used by the `atendente_core` service.

### Key Technologies

*   **RAG:** LangChain
*   **Package Manager:** Poetry
