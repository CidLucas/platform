# Vizu Qdrant Client

This library provides a standardized client for interacting with Qdrant in the Vizu ecosystem.

## Overview

The Vizu Qdrant Client simplifies the process of connecting to and interacting with the Qdrant vector database. It provides a pre-configured client that can be used by any service in the monorepo.

### Key Technologies

*   **Vector Database:** Qdrant
*   **Package Manager:** Poetry
