# Vizu LLM Service Client

This library provides a lightweight client for interacting with Large Language Model (LLM) and embedding services in the Vizu ecosystem.

## Overview

The Vizu LLM Service Client is a simple library that provides a convenient way to make requests to the LLM and embedding services. It uses `langchain-community` for interacting with Ollama and `requests` for other HTTP calls.

### Key Technologies

*   **LLM Interaction:** LangChain
*   **HTTP Client:** requests
*   **Package Manager:** Poetry
