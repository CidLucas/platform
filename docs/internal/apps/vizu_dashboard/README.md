# Vizu Dashboard

This directory contains the source code for the Vizu Dashboard, a React application built with Vite.

## Overview

The Vizu Dashboard is the main frontend for the Vizu application. It provides a user interface for interacting with the various backend services, including the conversational AI, data analytics, and file management features.

### Key Technologies

*   **Framework:** React
*   **Build Tool:** Vite
*   **UI Library:** Chakra UI
*   **Mapping:** Leaflet
*   **Charting:** Recharts
*   **Linting:** ESLint
*   **Package Manager:** npm

## Getting Started

To run the dashboard in a development environment, navigate to this directory and run the following command:

```bash
npm run dev
```

This will start the Vite development server.

## Building for Production

To build the application for production, run the following command:

```bash
npm run build
```

This will create a `dist` directory with the optimized production build.
