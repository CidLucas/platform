# Vizu Models

This library is the single source of truth for all Vizu data models.

## Overview

The Vizu Models library defines the data structures used throughout the Vizu application. It uses [SQLModel](https://sqlmodel.tiangolo.com/) to create models that are both SQLAlchemy models and Pydantic models.

This library ensures that data is consistent and validated across all services.

### Key Technologies

*   **Data Modeling:** SQLModel
*   **Package Manager:** Poetry
