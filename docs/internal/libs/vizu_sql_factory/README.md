# Vizu SQL Factory

This library provides a factory for creating SQL agents and tools for the Vizu application.

## Overview

The Vizu SQL Factory simplifies the process of creating and configuring SQL-based agents and tools. It uses `langchain-community` and `SQLAlchemy` to create custom SQL agents that can be used by the `atendente_core` service to interact with the database.

### Key Technologies

*   **SQL Agents:** LangChain
*   **Database:** SQLAlchemy
*   **Package Manager:** Poetry
